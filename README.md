Este projeto é uma solução completa para gestão de academias de artes marciais. Ele demonstra minha evolução técnica e meu foco em **Quality Assurance (QA)**, transformando um processo manual e simples em um software automatizado e profissional.

---

## 📈 Evolução do Produto (Mindset de QA)

Como aspirante a QA, utilizei este projeto para exercitar a melhoria contínua de software:

* **v1.0 (MVP):** Armazenamento em arquivos de texto simples (.txt) e interface nativa básica. (Arquivo `clientes_academia.txt` incluso no repositório).
* **v12.2 (Atual):** Migração para banco de dados Excel com integração **Pandas**, interface moderna em **Dark Mode** e dashboards dinâmicos.

---

## 🚀 Funcionalidades Principais

* **Dashboards de Métricas:** Visualização instantânea de total de alunos e pendências financeiras.
* **Gestão de Mensalidade:** Sistema de baixa de pagamento e "Novo Ciclo" para reset mensal automático.
* **Segurança de Dados:** Rotina de **Backup automático** com carimbo de data e hora (timestamp) antes de alterações críticas.
* **Busca Inteligente:** Filtro de pesquisa em tempo real para agilidade no atendimento.

---

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Python 3.x
* **Interface Gráfica:** CustomTkinter (UI moderna e responsiva)
* **Manipulação de Dados:** Pandas & Openpyxl
* **Distribuição:** PyInstaller (Compilação para executável Windows)

---

## 📁 Como rodar o projeto

1. Certifique-se de ter o Python instalado.
2. Instale as dependências: `pip install pandas customtkinter pillow openpyxl`
3. Execute o script: `python newgh.py`
